# 资源介绍

日程是日历系统的核心资源，代表在特定时间发生的活动。每个日程必须属于某个日历，包含时间、标题、地点等基本信息，可以邀请参与者、预订会议室、设置提醒。日程支持单次、循环和全天等类型，一个日程可以有多个参与者和多个会议室。

**日程属性**
- **id**: 日程的唯一标识符
- **calendar_id**: 所属日历ID
- **summary**: 日程标题
- **description**: 日程备注
- **start_time**: 开始时间
- **end_time**: 结束时间
- **locations**: 日程地址
- **status**: 日程状态（normal-正常状态，cancelled-已取消）
- **recurrence**: 重复规则，用于设置日程的重复方式（支持按天/周/月/年重复，指定次数或结束日期）
- **organizer**: 组织者信息
- **reminders**: 日程提醒
- **visibility**: 可见范围，控制日程是否公开（default-默认，public-公开，private-仅自己可见）
- **free_busy_status**: 忙闲状态（busy-忙碌，free-空闲）
- **online_meeting**: 在线会议信息，用于关联WPS会议。创建普通日程时此参数为可选，不需要在线会议功能时可以不填
- **recurring_event_id**: 所属的重复日期的id
- **original_start_time**: 所属重复日程被编辑的那一天
