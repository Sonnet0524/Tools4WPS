# 查询日历列表



**标签**：`日历` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/calendars          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 日历列表 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询日历信息(应用授权) `kso.calendar.read`</div><div style="margin-top: 5px;"></div><div>查询和管理日历信息(应用授权) `kso.calendar.readwrite`</div><div style="margin-top: 5px;"></div><div>查询日历信息(用户授权) `kso.calendar.read`</div><div style="margin-top: 5px;"></div><div>查询和管理日历信息(用户授权) `kso.calendar.readwrite`</div></div> |






## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">page_token</div> | `string` | 否 | 分页标记，第一次请求不需要传递此参数，后续请求需要传递上一次响应中的next_page_token | - 
| <div style="white-space: nowrap;">page_size</div> | `integer` | 否 | 每页返回的日历数量，默认为20，最大值为20 | -
