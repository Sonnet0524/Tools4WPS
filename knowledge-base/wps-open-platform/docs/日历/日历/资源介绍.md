# 资源介绍

日历是日程管理的基础容器。每个用户默认拥有一个主日历（通过 `primary` 标识访问，无法删除），也可创建多个自定义日历来分类管理不同类型的日程。一个日历可以包含多个日程，并可通过权限设置与其他用户共享。

**日历属性**
- **id**: 日历的唯一标识符
- **summary**: 日历标题
- **description**: 日历描述
- **type**: 日历类型（primary-主日历，normal-普通日历）
- **open_status**: 日历开放状态（仅主日历，private-私密，show_only_free_busy-仅显示忙闲，public-公开）
- **role**: 当前身份对该日历的权限级别（free_busy_reader-游客权限，reader-订阅者权限，writer-编辑者权限，owner-所有者权限）
- **is_deleted**: 日历是否删除
