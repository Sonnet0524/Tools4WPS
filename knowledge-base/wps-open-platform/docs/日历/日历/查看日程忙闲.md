# 查询主日历日程忙闲

查询指定用户的主日历忙闲信息，或查询指定会议室的忙闲信息，查询时间需要满足 end_time-start_time <= 7 天



**标签**：`日程忙闲` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/free_busy_list          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 查看日程忙闲， 查询时间需要满足 end_time-start_time <= 7 天  |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询日程(应用授权) `kso.calendar_events.read`</div><div style="margin-top: 5px;"></div><div>查询和管理日程信息(应用授权) `kso.calendar_events.readwrite`</div><div style="margin-top: 5px;"></div><div>查询日程(用户授权) `kso.calendar_events.read`</div><div style="margin-top: 5px;"></div><div>查询和管理日程信息(用户授权) `kso.calendar_events.readwrite`</div></div> |


## 请求头 (Headers)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">X-Kso-Id-Type</div> | `string` | 否 | 类型<br>  * **internal** - 内部<br>  * **external** - 外部<br> | `internal`, `external` 





## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">start_time</div> | `string` | 是 | 开始查询时间，需要url编码，RFC3339格式，例如：2024-01-15T14%3A30%3A00%2B08%3A00 | - 
| <div style="white-space: nowrap;">end_time</div> | `string` | 是 | 结束查询时间，需要url编码，RFC3339格式，例如：2024-01-15T14%3A30%3A00%2B08%3A00 | - 
| <div style="white-space: nowrap;">user_ids</div> | `array` | 否 | 用户id列表，可从通讯录接口获取，用于查询指定用户的忙闲状态 | - 
| <div style="white-space: nowrap;">room_ids</div> | `array` | 否 | 会议室id列表，可从会议室列表接口获取 | -
