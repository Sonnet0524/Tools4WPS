# 资源介绍

日程参与者代表被邀请参加某个日程的用户，支持企业内部用户和外部邮箱地址。参与者收到邀请后可以接受、拒绝、待定或关注日程。

**参与者属性**
- **id**: 参与者唯一标识ID
- **type**: 参与者类型（user-用户，group-用户组，third_party_email-第三方邮箱）
- **user_id**: 用户id（当type为user时有值）
- **group_id**: 用户组id（当type为group时有值）
- **third_party_email**: 第三方参与者邮箱（当type为third_party_email时有值）
- **response_status**: 响应状态（not_responded-未答复，declined-拒绝，accepted-接受，tentative-待定，follow-关注）
- **name**: 参与者名称
