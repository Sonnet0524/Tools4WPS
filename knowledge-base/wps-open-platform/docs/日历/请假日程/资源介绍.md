# 资源介绍

请假日程（Timeoff Event）是一种特殊类型的日程，专门用于记录用户的请假、休假、病假等非工作时间，默认为"忙碌"状态，并且还会在协作中的对应请假时间段内显示对应的请假中标识。

**请假日程属性**
- **timeoff_event_id**: 请假日程的唯一标识符
- **user_id**: 用户ID
- **timeoff_type**: 请假类型（business_trip-出差，annual_leave-休假，time_off-请假，external_visit-外访）
- **start_time**: 请假开始时间
- **end_time**: 请假结束时间
- **title**: 日程标题
- **description**: 详细说明或备注
