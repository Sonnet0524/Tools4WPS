# 日历权限变更

## 事件描述

日历权限发生变化时，包括添加、修改、删除权限时触发此消息。

## 订阅说明

| **事件主体** | **kso.calendar.permission**                                                            |
|:---------|:---------------------------------------------------------------------------------------|
| **权限要求** | 查询日历信息 `kso.calendars.read`                                                            |
| **解密方式** | 详见 [解密算法](/app-integration-dev/wps365/server/event-subscription/security-verification) |

### 事件体（解密前）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**         | **参数类型** | **说明**                                           |
|:---------------|:---------|:-------------------------------------------------|
| topic          | string   | 消息主题 kso.calendar.permission                     |
| operation      | string   | 消息变更动作<br/>update：更新<br/>create：创建<br/>delete：删除 |
| time           | integer  | 时间（秒为单位的时间戳）                                     |
| nonce          | string   | iv 向量（解密时使用）                                     |
| signature      | string   | 消息签名                                             |
| encrypted_data | string   | 消息变更的加密字段                                        |

### 事件体示例（解密前）

<!-- 为了保持文档格式统一，请使用 2 个空格作为 json 的缩进符 -->
```json
{

    "topic": "kso.calendar.permission",

    "operation": "update",

    "time": 1704074400,

    "nonce": "71***********7",

    "signature": "w6**********6Q",

    "encrypted_data": "B7**********iA=="

}
```

### 事件体数据（解密后）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**               | **参数类型**      | **是否必带** | **说明**                                                                             |
|:---------------------|:--------------|:---------|:-----------------------------------------------------------------------------------|
| company_id           | string        | 是        | 企业 ID                                                                              |
| calendar_permissions | array[object] | 是        | 日历权限变更信息数组                                                                         |
| ∟ id                 | string        | 是        | 日历权限 ID                                                                            |
| ∟ calendar_id        | string        | 是        | 日历 ID                                                                              |
| ∟ user_id            | string        | 是        | 用户 ID                                                                              |
| ∟ role               | string        | 是        | 日历权限级别<br/>free_busy_reader：游客权限<br/>reader：订阅者权限<br/>writer：编辑者权限<br/>owner：管理员权限 |

### 事件体数据示例（解密后）

<!-- 为了保持文档格式统一，请使用 2 个空格作为 json 的缩进符 -->

```json
{

    "company_id": "123456",

    "calendar_permissions": [

    {

        "id": "perm_001",

        "user_id": "user_456",

        "role": "writer"

    },

    {

        "id": "perm_002",

        "calendar_id": "cal_123",

        "user_id": "user_789",

        "role": "reader"

    }]

}
```
