# 智能体应用详情查询

查询智能体应用的详细信息，包括使用智能体应用前需要申请哪些scope能力的依赖，应用包含哪些工具列表，应用名称



**标签**：`智能体应用` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/devhub/apps/{app_id}/detail          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 智能体应用详情查询 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>智能体应用查询和创建(应用授权) `kso.devhub_app.readwrite`</div><div style="margin-top: 5px;"></div><div>智能体应用查询和创建(用户授权) `kso.devhub_app.readwrite`</div></div> |
