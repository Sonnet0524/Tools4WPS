# 查询会话历史消息

会话是限定上下文内容的范围，在一个会话中，调用方和智能体可以进行多次对话，每次对话的时候，智能体会自动带上当前会话里面的历史消息



**标签**：`会话` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/devhub/apps/{app_id}/sessions/messages          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 查询会话历史消息 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>智能体会话管理(用户授权) `kso.devhub_session.readwrite`</div></div> |


## 请求头 (Headers)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">Content-Type</div> | `string` | 是 | 使用：'application/json'  | - 
| <div style="white-space: nowrap;">X-Kso-Date</div> | `string` | 是 | RFC1123 格式的日期，例: 'Wed, 23 Jan 2013 06:43:08 GMT'  | - 
| <div style="white-space: nowrap;">X-Kso-Authorization</div> | `string` | 是 | KSO-1 签名值 | - 
| <div style="white-space: nowrap;">Authorization</div> | `string` | 是 | 授权凭证，格式为：'Bearer {access_token}' | - 



## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">app_id</div> | `string` | 是 |  | - 



## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">session_id</div> | `string` | 否 | 会话id | - 
| <div style="white-space: nowrap;">with_event</div> | `boolean` | 否 | 是否携带调试的信息 | - 
| <div style="white-space: nowrap;">next_page_token</div> | `string` | 否 | 分页token | - 
| <div style="white-space: nowrap;">page_size</div> | `integer` | 否 | 分页大小 | -
