# 常见问题

## 1.调用查询块后，结果里出现的rangeMarkBegin（End）是什么？

是[区间信息](/app-integration-dev/wps365/server/airpage/data_structure#3%E5%8C%BA%E9%97%B4%E4%BF%A1%E6%81%AF)，用于表示评论的起始结束位置。当调用更新文档块的update\_content操作时，若希望保留评论，可以将查询得到的rangeMark填入content中。注意：当调用的操作（如创建文档块、删除文档块等）涉及index时，需要忽略rangeMark，因为rangeMark并不是真实存在的节点。

## 2.appComponent节点如何保留？

appComponent的数据来源复杂，因此目前无法通过创建文档块插入文档。但可以在调用update\_content操作时，将查询得到的appComponent填入content中，避免文档已有的appComponent丢失。
