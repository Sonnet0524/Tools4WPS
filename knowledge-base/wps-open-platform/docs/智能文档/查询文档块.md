# 查询文档块

查询文档中的块

## 请求说明

| **请求地址** | [**https://openapi.wps.cn/v7/airpage/{file\_id}/blocks**](https://openapi.wps.cn/v7/airpage/%7Bfile_id%7D/blocks)                                                                                                                                                                       |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **请求方法** | POST                                                                                                                                                                                                                                                                                       |
| **签名方式** | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description)                                                                                                                                                                           |
| **权限要求** | 管理智能文档(应用授权) <span style="font-size: 1em;">`kso.airpage.readwrite`</span>查询智能文档(应用授权) <span style="font-size: 1em;">`kso.airpage.read`</span>管理智能文档(用户授权) <span style="font-size: 1em;">`kso.airpage.readwrite`</span>查询智能文档(用户授权) <span style="font-size: 1em;">`kso.airpage.read`</span> |

## 请求参数

| **名称**   | **参数类型** | **是否必填** | **位置** | **说明**                                                                                                         |
| -------- | -------- | -------- | ------ | -------------------------------------------------------------------------------------------------------------- |
| file\_id | string   | 是        | path   | 文档的id                                                                                                          |
| arg      | string   | 是        | body   | json结构体的base64编码，具体见[请求体](/app-integration-dev/wps365/server/airpage/query_block#%E8%AF%B7%E6%B1%82%E4%BD%93) |

## 请求体

```
Request:
  type: object
  required:
    - arg
  properties:
    arg:
      type: object
      required:
        - blockId
      properties:
        blockId:
          type: string
          description: 块ID，为"doc"查询文档块
```

## 请求体示例

此处示例为base64编码前的数据

```
{
    "arg": {
        "blockId": "xxx"
    }
}
```

## 响应体

```
response:
  type: object
  properties:
    code:
      type: integer
      description: 正常情况下为0
    msg:
      type: string
      description: 错误信息
    data:
      type: object
      properties:
        result:
        type: object
        properties:
          blocks:
            type: array
            items:
              type: object
            description: 块结果
          version:
            type: integer
            description: 文档数据版本
```
