# 更新文档块

对指定块进行更新操作

## 请求说明

| **请求地址** | [**https://openapi.wps.cn/v7/airpage/{file\_id}/blocks/update**](https://openapi.wps.cn/v7/airpage/%7Bfile_id%7D/blocks/update)                 |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| **请求方法** | POST                                                                                                                                               |
| **签名方式** | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description)                                   |
| **权限要求** | 管理智能文档(应用授权) <span style="font-size: 1em;">`kso.airpage.readwrite`</span>管理智能文档(用户授权) <span style="font-size: 1em;">`kso.airpage.readwrite`</span> |

## 请求参数

| **名称**   | **参数类型** | **是否必填** | **位置** | **说明**                                                                                                          |
| -------- | -------- | -------- | ------ | --------------------------------------------------------------------------------------------------------------- |
| file\_id | string   | 是        | path   | 文档的id                                                                                                           |
| arg      | string   | 是        | body   | json结构体的base64编码，具体见[请求体](/app-integration-dev/wps365/server/airpage/update_block#%E8%AF%B7%E6%B1%82%E4%BD%93) |

## 请求体

由具体操作决定

以下示例均为base64编码前的内容

### 具体操作列表：

### update\_content

更新块的内容

```
request:
  type: object
  required:
    - arg
  properties:
    arg:
      type: object
      required:
        - blockId
        - operation
        - content
      properties:
        blockId:
          type: string
          description: 块ID，通过查询文档块获取，为"doc"表示更新文档内容
        operation:
          type: string
          enum: ['update_content']
          description: 固定为"update_content"
        content:
          type: array
          items:
            type: object
          description: 块的内容，不包含块自身
```

### 示例：

```
{
    "arg": {
        "operation": "update_content",
        "blockId": "xxx",
        "content": [...]
    }
}
```

### update\_attrs

更新块的属性

```
request:

        operation:
```

* <span style="font-size: 14px;">目前不支持doc、appComponent、lockBlock设置属性</span>
* <span style="font-size: 14px;">tableCell的colspan和rowspan属性请通过表格相关操作设置。</span>

```
request:

      properties:
```

```
        ]
    }
```

### merge\_table\_cells

合并单元格，blockId对应块必须为table

```
request:
  type: object
  required:
    - arg
  properties:
    arg:
      type: object
      required:
        - blockId
        - operation
        - rowSpan
        - colSpan
      properties:
        blockId:
          type: string
          description: 块ID，对应块必须是table
        operation:
          type: string
          enum: ['merge_table_cells']
          description: 固定为"merge_table_cells"
        startRow:
          type: integer
          default: 0
          description: 起始行号
        startCol:
          type: integer
          default: 0
          description: 起始列号
        rowSpan:
          type: integer
          minimum: 1
          description: 合并行数，为1代表不合并（rowSpan和colSpan不可同时为1）
        colSpan:
          type: integer
          minimum: 1
          description: 合并列数，为1代表不合并（rowSpan和colSpan不可同时为1）
```

### 示例：

```
{
    "arg": {
        "operation": "merge_table_cells",
        "blockId": "xxx",
        "startRow": 0,
        "startCol": 0,
        "rowSpan": 2,
        "colSpan": 3
    }
}
```

### replace\_anchor

替换blockId对应块内的指定占位节点

```
request:
  type: object
  required:
    - arg
  properties:
    arg:
      type: object
      required:
        - blockId
        - operation
        - anchorId
        - content
      properties:
        blockId:
          type: string
          description: 块ID，通过查询文档块获取，为"doc"表示替换全文符合要求的占位节点
        operation:
          type: string
          enum: ['replace_anchor']
          description: 固定为"replace_anchor"
        anchorId:
          type: string
          description: 占位节点id
        content:
          type: object
          description: 替换块，type必须与占位节点的aimType一致
```

### 示例：

```
{
    "arg": {
        "operation": "replace_anchor",
        "blockId": "xxx", // 表示仅替换该block下的blockAnchor，为“doc”替换全文
        ”anchorId": "123",
        "content": {
            "type": "picture",
            "attrs": {} // 需填入完整属性
        }
    }
}
```

### replace\_feature

替换blockId对应块内的指定特性元素

```
request:
  type: object
  required:
    - arg
  properties:
    arg:
      type: object
      required:
        - blockId
        - operation
        - source
        - target
      properties:
        blockId:
          type: string
          description: 块ID，通过查询文档块获取，为"doc"表示替换全文符合要求的特性元素
        operation:
          type: string
          enum: ['replace_feature']
          description: 固定为"replace_feature"
        source:
          type: object
          description: |
            指定特性元素，目前可包含以下类型，并且需要指定相关属性：
            - WPSUser: userId
            - WPSDocument: wpsDocumentId
            - schedule: scheduleId
        target:
          type: object
          description: 替换特性元素，type需要与source一致
```

### 示例：

```
{
    "arg": {
        "operation": "replace_feature",
        "blockId": "xxx", // 表示仅替换该block下的feature，为“doc”替换全文
        "source": {
            "type": "WPSUser",
            "attrs": {
                "userId": "123",
                "name": "张三"
            }
        },
        "target": {
            "type": "WPSUser",
            "attrs": {
                "userId": "456",
                "name": "李四"
            }
        }
    }
}
```

## 响应体

```
response:
  type: object
  properties:
    code:
      type: integer
      description: 正常情况下为0
    msg:
      type: string
      description: 错误信息
    data:
      type: object
      properties:
        result:
        type: object
        properties:
          blocks:
            type: array
            items:
              type: object
            description: 更新后的块
          version:
            type: integer
            description: 文档数据版本
```
