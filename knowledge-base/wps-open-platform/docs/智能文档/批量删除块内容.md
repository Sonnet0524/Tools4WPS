# 批量删除块内容

批量删除指定块的内容

## 请求说明

| **请求地址** | [**https://openapi.wps.cn/v7/airpage/{file\_id}/blocks/batch\_delete**](https://openapi.wps.cn/v7/airpage/%7Bfile_id%7D/blocks/batch_delete)    |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| **请求方法** | POST                                                                                                                                               |
| **签名方式** | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description)                                   |
| **权限要求** | 管理智能文档(应用授权) <span style="font-size: 1em;">`kso.airpage.readwrite`</span>管理智能文档(用户授权) <span style="font-size: 1em;">`kso.airpage.readwrite`</span> |

## 请求参数

| **名称**   | **参数类型** | **是否必填** | **位置** | **说明**                                                                                                                                    |
| -------- | -------- | -------- | ------ | ----------------------------------------------------------------------------------------------------------------------------------------- |
| file\_id | string   | 是        | path   | 文档的id                                                                                                                                     |
| arg      | string   | 是        | body   | json结构体的base64编码，具体见[请求体说明](/app-integration-dev/wps365/server/airpage/batch_delete_block#%E8%AF%B7%E6%B1%82%E4%BD%93%E8%AF%B4%E6%98%8E) |

## 请求体说明

[删除块内容的请求体](/app-integration-dev/wps365/server/airpage/delete_block#%E8%AF%B7%E6%B1%82%E4%BD%93)的数组形式

## 请求体示例

此处示例为base64编码前的数据

```
{
  "arg": [
    {
      "blockId": "doc",
      "startIndex": 1,
      "endIndex": 2
    }
  ]
}
```

## 响应体

```
response:
  type: object
  properties:
    code:
      type: integer
      description: 正常情况下为0
    msg:
      type: string
      description: 错误信息
    data:
      type: object
      properties:
        success_list:
          type: array
          items:
            type: object
            properties:
              blockId:
                type: string
              version:
                type: integer
        fail_list:
          type: array
          items:
            type: object
```

## 响应体示例

此处示例为base64编码前的数据

```
{
  "code": 0,
  "msg": "",
  "data": {
    "success_list": [
      {
        "blockId": "doc",
        "version": 27
      }
    ],
    "fail_list": []
  }
}
```
