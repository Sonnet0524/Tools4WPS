# 分页查询公告可见范围

根据公告ID分页查询可见的企业/部门/用户列表



**标签**：`公告管理` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/announce/announces/{announce_id}/visibilities          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 根据公告ID分页查询可见的企业/部门/用户列表 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询公告(应用授权) `kso.announce.read`</div></div> |


## 请求头 (Headers)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">X-Kso-Id-Type</div> | `string` | 否 | 类型<br>  * **internal** - 内部<br>  * **external** - 外部<br> | `internal`, `external` 



## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">announce_id</div> | `string` | 是 | 公告ID（字符串格式） | - 



## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">page_size</div> | `integer` | 是 | 分页大小（范围1-100，默认20） | - 
| <div style="white-space: nowrap;">page_token</div> | `string` | 否 | 分页token（首次请求不传） | - 
| <div style="white-space: nowrap;">with_total</div> | `boolean` | 否 | 是否返回总数 | -
