# 公告变更事件

## 事件描述

企业公告发送/更新/删除/撤回时通知。

## 订阅说明

| **事件主体** | kso.xz.announce                                                                                              |
| :----------- |:-------------------------------------------------------------------------------------------------------------|
| **权限要求** | 查询公告 `kso.announce.read`                                                                                     |
| **解密方式** | 详见 [解密算法](/app-integration-dev/wps365/server/event-subscription/security-verification) |

### 事件体（解密前）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**       | **参数类型** | **说明**                                                     |
| :------------- | :----------- |:-----------------------------------------------------------|
| topic          | string       | 消息主题 `kso.xz.announce`                                     |
| operation      | string       | 消息变更动作<br/>`create`：发布；`update`：更新；`delete`：删除；`recall`：撤回 |
| time           | integer      | 时间（秒为单位的时间戳）                                               |
| nonce          | string       | iv 向量（解密时使用）                                               |
| signature      | string       | 消息签名                                                       |
| encrypted_data | string       | 消息变更的加密字段                                                  |

### 事件体示例（解密前）

<!-- 为了保持文档格式统一，请使用 2 个空格作为 json 的缩进符 -->

```json
{
  "topic": "kso.xz.announce",
  "operation": "create",
  "time": 1704074400,
  "nonce": "71***********7",
  "signature": "w6**********6Q",
  "encrypted_data": "B7**********iA=="
}
```

### 事件体数据（解密后）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**         | **参数类型**      | **是否必带** | **说明**                                                                                    |
|:---------------|:--------------|:---------|:------------------------------------------------------------------------------------------|
| announce_event_type          |   string   | 是        | 事件类型：announce_event（公告事件）、announce_security_level_event（密级事件）、announce_group_event：（分组事件） |
| event_announce                |   object  | 否        | 公告事件主体数据                                                                                  |
| ∟ id                         |   string   | 否       | 公告id                                                                                      |
| ∟ is_top                     |  boolean   |  否      | 置顶状态（true=置顶，false=非置顶）                                                                   |
| event_security_level          |   object   |   否       | 密级事件                                                                                      |
| ∟ id                         |   string   |   否       | 密级id                                                                                      |
| ∟ mobile_visible             |  boolean   |   否       | 移动端可见性（true=可见，false=隐藏）                                                                  |
| event_group                   |   object  |   否       | 公告分组信息                                                                                    |
| ∟ id                         |   string   |   否       | 分组id                                                                                      |
| ∟ name                       |   string   |   否       | 分组名称                                                                                      |


### 事件体数据示例（解密后）

<!-- 为了保持文档格式统一，请使用 2 个空格作为 json 的缩进符 -->

```json
{
  "announce_event_type": "announce_event",

    "id": "2",
    "mobile_visible": true 
  },
  "event_group": {
    "id": "1",
    "name": "分组名称"
  }
}
```
