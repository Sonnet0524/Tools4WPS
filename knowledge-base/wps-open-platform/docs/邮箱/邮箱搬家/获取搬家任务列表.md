# 获取搬家任务列表



**标签**：`邮箱搬家` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/mail_migrations/{migration_id}/tasks          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 获取搬家任务列表 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询和管理邮箱搬家(应用授权) `kso.mail_migration.readwrite`</div><div style="margin-top: 5px;"></div><div>查询邮箱搬家(应用授权) `kso.mail_migration.read`</div></div> |




## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">migration_id</div> | `string` | 是 | 搬家作业ID，填入current获取当前企业正在进行作业的搬家任务 | - 



## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">state</div> | `string` | 否 | 搬家任务状态，默认全部 | `pending`, `running`, `paused`, `failed`, `cancelled`, `success`, `closed` 
| <div style="white-space: nowrap;">page_size</div> | `integer` | 否 | 分页大小（范围1-100，默认20） | - 
| <div style="white-space: nowrap;">page_token</div> | `string` | 否 | 翻页token（第一次请求不传） | -
