# 获取指定邮件



**标签**：`邮件` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/mailboxes/{mailbox_id}/folders/{folder_id}/messages/{message_id}          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 从用户特定邮箱提取一封邮件，支持用户身份。 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询和管理邮件(用户授权) `kso.mail.readwrite`</div><div style="margin-top: 5px;"></div><div>查询邮件(用户授权) `kso.mail.read`</div></div> |




## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">mailbox_id</div> | `string` | 是 |  | - 
| <div style="white-space: nowrap;">folder_id</div> | `string` | 是 |  | - 
| <div style="white-space: nowrap;">message_id</div> | `string` | 是 |  | - 



## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">with_ocr_body</div> | `boolean` | 否 | 是否需要OCR识别后的邮件体。默认返回未经OCR识别的邮件体。 | - 
| <div style="white-space: nowrap;">with_importance_flag</div> | `boolean` | 否 | 是否需要邮件重要性标识。默认不返回importance_flag字段 | -
