# 获取邮箱联系人列表



**标签**：`邮箱联系人` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/mail_contacts          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 获取在当前企业范围内邮箱联系人列表 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询和管理邮箱联系人(应用授权) `kso.mail_contact.readwrite`</div><div style="margin-top: 5px;"></div><div>查询邮箱联系人(应用授权) `kso.mail_contact.read`</div></div> |






## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">page_size</div> | `integer` | 否 | 分页大小 | - 
| <div style="white-space: nowrap;">page_token</div> | `string` | 否 | 翻页token,获取首页无需提供 | - 
| <div style="white-space: nowrap;">with_total</div> | `boolean` | 否 | 是否返回total字段,默认不返回 | - 
| <div style="white-space: nowrap;">contact_ids</div> | `array` | 否 | 邮箱联系人ID列表，传入多个时以英文逗号','分隔。指定id获取时会忽略分页参数，且不会进行分页返回。 | -
