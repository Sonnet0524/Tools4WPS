# 资源介绍

会议室设置是针对每个会议室的配置参数，用于定义使用规则和限制。设置类型包括预约限制（时间段、时长、提前预约时间等）、审批设置（是否需要审批、审批人等）和访问范围（可预约的成员范围）。每个会议室对应一套设置参数。

**会议室设置属性**
- **room_id**: 会议室ID
- **booking_limit**: 是否开启预约限制
- **booking_time**: 预约时间范围（格式为hh:mm-hh:mm，如08:00-24:00）
- **booking_duration**: 单次预约的最长时长限制（单位分钟）
- **booking_preview**: 最早可提前预定时间（单位天）
- **approve_time**: 需要审核的会议时长阈值（单位分钟）
- **access_scope**: 可预约会议室的成员范围设置
- **approval_setting**: 会议室审批设置
