# 资源介绍

会议室层级用于组织会议室的树状结构。企业可以按楼层、区域、园区等维度对会议室进行分类组织。层级支持多级嵌套（5级，且会议室必须挂在第五级会议室层级上），每个层级可以包含子层级和会议室，根层级的 parent_id 为 0。

**层级属性**
- **id**: 层级ID
- **name**: 层级名称（如"1F"、"东区"、"总部"）
- **parent_id**: 父层级ID（当层级为企业根目录时，parent_id为0）
- **path**: 会议室层级路径，从根层级到本层级的id路径，用于表示完整的层级关系
- **has_child**: 是否有子层级（true表示当前层级下还有子层级）
