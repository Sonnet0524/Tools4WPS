# 资源介绍

会议室预约记录了会议室的预订信息，包括预约时间、预约人、预约状态等。预约状态包括预约成功、预约失败、审批中，签到状态包括未签到、已签到、未签到释放、无需签到。预约可以关联到日历日程和视频会议，一个会议室可以有多个预约记录。

**预约属性**
- **id**: 预约ID
- **room_id**: 会议室ID
- **room_name**: 会议室名称
- **summary**: 主题
- **user**: 预约人信息（包含id和user_name）
- **start_time**: 预约开始时间（Unix时间戳，毫秒）
- **end_time**: 预约结束时间（Unix时间戳，毫秒）
- **create_time**: 预约日程的创建时间
- **result**: 预约结果（success-预约成功，failed-预约失败，approving-审批中）
- **event_id**: 日程ID
- **calendar_id**: 日历ID
- **dept_name**: 预定人所属部门名称
- **invite_count**: 邀约人数
- **accept_count**: 接受邀约人数
- **fail_reason**: 预约失败原因
- **sign_in_status**: 会议室签到状态（wait-未签到，success-已签到，release-未签到释放，empty-无需签到）
- **sign_time**: 会议室签到时间
- **is_recurring**: 是否重复日程
- **delete_time**: 删除时间
