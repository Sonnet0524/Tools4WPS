# 会议室变更

## 事件描述

会议室资源变化时，包括 创建、修改、删除时触发此消息。

## 订阅说明

| **事件主体** | kso.meeting_room                                                                                       |
| :----------- | :--------------------------------------------------------------------------------------------- |
| **权限要求** | 查询会议室信息 `kso.meeting_rooms.read`                                                              |
| **解密方式** | 详见 [解密算法](/app-integration-dev/wps365/server/event-subscription/security-verification) |

### 事件体（解密前）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**       | **参数类型** | **说明**                                                       |
| :------------- | :----------- | :------------------------------------------------------------- |
| topic          | string       | 消息主题 `kso.meeting_room`                                            |
| operation      | string       | 消息变更动作<br/>`create`：创建；`update`：更新；`delete`：删除 |
| time           | integer      | 时间（秒为单位的时间戳）                                       |
| nonce          | string       | iv 向量（解密时使用）                                          |
| signature      | string       | 消息签名                                                       |
| encrypted_data | string       | 消息变更的加密字段                                             |

### 事件体示例（解密前）

<!-- 为了保持文档格式统一，请使用 2 个空格作为 json 的缩进符 -->

```json
{
  "topic": "kso.meeting_room",
  "operation": "create|update|delete",
  "time": 1704074400,
  "nonce": "71***********7",
  "signature": "w6**********6Q",
  "encrypted_data": "B7**********iA=="
}
```

### 事件体数据（解密后）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**         | **参数类型**      | **是否必带** | **说明**                   |
|:---------------|:--------------|:---------|:-------------------------|
| company_id     | string        | 是        | 企业 id                  |
| rooms          | array[object] | 是        | 会议室变更信息数组          |
| ∟ id           | string        | 是        | 会议室 id                 |
| ∟ level_id     | string        | 是        | 会议室层级 id             |
| ∟ name         | string        | 是        | 会议室名称                |
| ∟ remark       | string        | 是        | 备注                     |
| ∟ capacity     | integer       | 是        | 会议室容量(单位/人) |
| ∟ disable      | boolean       | 是        | 会议室是否禁用 |
| ∟ devices      | array[string] | 是        | 设备名列表 |

### 事件体数据示例（解密后）

<!-- 为了保持文档格式统一，请使用 2 个空格作为 json 的缩进符 -->

```json
{
    "company_id": "A*****6",
    "rooms": [
        {
            "id": "37829",
            "level_id": "121",
            "name": "101会议室",
            "remark": "",
            "capacity": 1,
            "disable": false,
            "devices": [
                "电视"
            ]
        }
    ]
}
```
