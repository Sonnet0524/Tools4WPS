# 资源介绍

会议室是企业的线下会议资源，代表可供预订的物理会议空间。每个会议室有名称、所属层级、容量、设备列表等属性。会议室归属于某个层级，有独立的设置（预约规则等），可设置白名单或黑名单，一个会议室可以有多个预约记录。

**会议室属性**
- **id**: 会议室id
- **level_id**: 会议室层级id
- **name**: 会议室名称
- **remark**: 会议室备注
- **capacity**: 会议室容量（可容纳的最大人数）
- **disable**: 当前会议室禁用状态（true-已被禁用不可预约，false-可正常使用）
- **device_list**: 设备列表（包含会议室配备的设备详情）
- **tag_list**: 会议室标签列表（用于分类和筛选）
