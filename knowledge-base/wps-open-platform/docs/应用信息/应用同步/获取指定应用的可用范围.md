# 获取指定应用的可用范围

获取指定企业自建应用的可用范围



**标签**：`应用` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/developer/apps/{app_id}/subjects          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 获取指定企业自建应用的可用范围 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询应用可用范围信息(应用授权) `kso.app_subject.read`</div><div style="margin-top: 5px;"></div><div>范围查询和管理应用可用范围(应用授权) `kso.app_subject.readwrite`</div></div> |


## 请求头 (Headers)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">X-Kso-Id-Type</div> | `string` | 否 |  | `internal`, `external` 



## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">app_id</div> | `string` | 是 | 应用id | - 



## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">page_size</div> | `integer` | 是 | 分页大小 | - 
| <div style="white-space: nowrap;">page_token</div> | `string` | 否 | 分页标记，第一次请求不填，表示从头开始遍历 | - 
| <div style="white-space: nowrap;">with_total</div> | `boolean` | 否 | 是否返回total字段，默认不返回 | - 
| <div style="white-space: nowrap;">is_blacklist</div> | `boolean` | 否 | 是否返回黑名单 | -
