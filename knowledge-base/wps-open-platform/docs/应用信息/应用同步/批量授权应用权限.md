# 批量授权应用权限

批量授权应用权限，暂只支持传入基础单点登录需要的scope：kso.user_base.read



**标签**：`应用管理` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/developer/apps/{app_id}/scopes/batch_grant          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `POST`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 批量授权应用权限，暂只支持传入基础单点登录需要的scope：kso.user_base.read |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询和管理应用版本信息(应用授权) `kso.app_version.readwrite`</div></div> |




## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">app_id</div> | `string` | 是 | 应用的id | -
