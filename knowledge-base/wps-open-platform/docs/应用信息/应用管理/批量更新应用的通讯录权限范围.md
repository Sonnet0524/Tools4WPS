# 批量更新应用的通讯录权限范围

    批量更新应用的通讯录权限范围，支持新增和删除，仅支持企业自建应用。
  注意：
  1. 私网最多支持配置 **1500** 项内容；公网支持白名单项，目前白名单项最多支持配置 **1500** 项内容，非白名单项最多支持配置 **200** 项内容；
  2. 设置【全部成员】和【与可用范围一致】为全量更新方式；设置【权限范围列表】为批量更新方式。

    

**标签**：`应用管理`

| 字段          | 值                  |
|--------------|---------------------|

| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
