# 获取用户可用的应用

> 获取主体可用范围内的应用列表。



**标签**：`应用` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/developer/visible_apps          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 获取主体可用范围内的应用列表 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>读取应用信息权限(应用授权) `kso.apps.read`</div></div> |






## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">id</div> | `string` | 是 | 主体id | - 
| <div style="white-space: nowrap;">type</div> | `string` | 是 | 主体类型 | `company`, `dept`, `user`, `group` 
| <div style="white-space: nowrap;">page_size</div> | `integer` | 是 | 分页大小 | - 
| <div style="white-space: nowrap;">page_token</div> | `string` | 否 | 分页标记 | - 
| <div style="white-space: nowrap;">with_total</div> | `boolean` | 否 | 是否返回total字段 默认不返回 | -
