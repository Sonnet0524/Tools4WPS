# 删除企业应用可用范围

> 批量移除当前应用身份所属企业下的应用可用范围，公网目前只支持白名单项。



**标签**：`应用` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/developer/apps/{app_id}/subjects/batch_remove          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `POST`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 批量移除当前应用身份所属企业下的应用可用范围，移除内容包括白名单项和黑名单项。<br>  注意：<br>  1. 移除不存在的项不会报错；<br>  2. 和 批量更新企业应用可用范围 区别在于，此接口为指定移除，后者是全量更新。<br>   |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询和管理应用可用范围(应用授权) `kso.app_subject.readwrite`</div></div> |


## 请求头 (Headers)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">X-Kso-Id-Type</div> | `string` | 否 | 类型<br>  * **internal** - 内部<br>  * **external** - 外部<br> | `internal`, `external` 



## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">app_id</div> | `string` | 是 | 应用id | -
