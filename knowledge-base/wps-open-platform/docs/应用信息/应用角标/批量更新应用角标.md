# 批量更新应用角标

> 批量更新应用角标信息，同步到协作，让信息及时通知到用户。



**标签**：`应用角标` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/developer/app_badges/batch_update          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `POST`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 批量更新应用角标信息，同步到协作，让信息及时通知到用户。<br>  应用角标是应用的一种信息通知能力，类似于“即时消息”的未读消息数量标识。通过这一功能，应用图标的右上角会显示红色数字角标，提醒用户应用内有需要关注的信息或待处理的任务。<br>  除此之外，聚合可见应用的角标，显示在工作台图标的右上角，提醒用户需要关注的信息或待处理的总量。 角标支持展示的数字范围是 1 到 99，说明如下：<br>  1. 当应用角标的待办数量小于等于 99 时，显示为红色数字。<br>  2. 当应用角标的待办数量大于 99 时，显示为红点。<br>  3. 当应用角标的待办数量为0时，不展示应用角标。<br>   |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>修改应用信息权限(应用授权) `kso.apps.readwrite`</div></div> |
