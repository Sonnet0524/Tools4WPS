# 获取应用管理员管理范围

> 获取管理和审核应用的管理员的管理范围，目前仅支持获取可管理的部门信息。注意，该应用管理员并非应用成员配置中的应用管理员，而是可进入企业管理后台对应用进行管理和审核的企业管理员。



**标签**：`应用管理` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/developer/get_app_admin_scope          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 获取管理和审核应用的管理员的管理范围，目前仅支持获取可管理的部门信息。注意，该应用管理员并非应用成员配置中的应用管理员，而是可进入企业管理后台对应用进行管理和审核的企业管理员。 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询应用管理信息(应用授权) `kso.app_admin.read`</div></div> |


## 请求头 (Headers)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">X-Kso-Id-Type</div> | `string` | 否 | 类型<br>  * **internal** - 内部<br>  * **external** - 外部<br> | `internal`, `external` 





## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">user_id</div> | `string` | 是 | 管理员对应的用户ID，支持user_id或ex_user_id | -
