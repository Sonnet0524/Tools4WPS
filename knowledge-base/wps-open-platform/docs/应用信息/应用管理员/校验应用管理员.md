# 校验应用管理员

> 校验用户是否为应用管理和审核的管理员。注意，该应用管理员并非应用成员配置中的应用管理员，而是可进入企业管理后台对应用进行管理和审核的企业管理员。



**标签**：`应用管理` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/developer/app_admins/{user_id}/check          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 校验用户是否为应用管理和审核的管理员.注意，该应用管理员并非应用成员配置中的应用管理员，而是可进入企业管理后台对应用进行管理和审核的企业管理员。 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询应用管理信息(应用授权) `kso.app_admin.read`</div></div> |
