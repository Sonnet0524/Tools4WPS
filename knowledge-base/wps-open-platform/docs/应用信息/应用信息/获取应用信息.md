# 获取指定应用详情

> 提供获取应用信息的功能。通过调用此接口，开发者可以获取指定应用的应用名称、应用ID等关键信息。



**标签**：`应用` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/developer/apps/{app_id}          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 获取应用信息 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询应用版本信息(应用授权) `kso.app_version.read`</div><div style="margin-top: 5px;"></div><div>查询和管理应用版本信息(应用授权) `kso.app_version.readwrite`</div></div> |


## 请求头 (Headers)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">X-Kso-Id-Type</div> | `string` | 否 | 类型<br>  * **internal** - 内部<br>  * **external** - 外部<br> | `internal`, `external` 



## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">app_id</div> | `string` | 是 |  | - 



## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">lang</div> | `string` | 否 | 优先展示语言，指定获取应用在该语言下的信息。<br>            可选值：<br>            * zh-CN: 中文<br>            * en-US: 英文<br>            * ja-JP: 日文<br>            * zh-HK: 繁体中文-香港<br>            * th-TH: 泰文<br>            默认值：zh-CN。<br>            若未配置应用i18n信息，则显示中文。 | `zh-CN`, `en-US`, `ja-JP`, `zh-HK`, `th-TH`
