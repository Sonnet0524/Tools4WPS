# 转让应用所有者

> 将自建应用的所有者变更为另外一个人, 如果是用户授权token, 必须保证用户是应用的所有者。



**标签**：`应用` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/developer/apps/{app_id}/change_owner          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `POST`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 将自建应用的所有者变更为另外一个人, 如果是用户授权token, 必须保证用户是应用的所有者 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>修改应用信息权限(应用授权) `kso.apps.readwrite`</div><div style="margin-top: 5px;"></div><div>修改应用信息权限(用户授权) `kso.apps.readwrite`</div></div> |
