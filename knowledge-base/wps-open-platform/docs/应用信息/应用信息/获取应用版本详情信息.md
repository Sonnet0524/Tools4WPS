# 获取应用版本信息

> 获取应用版本详情信息, 如果是历史版本: 返回历史版本的全量记录；如果是未审批版本: 返回未审批版本的全量记录。



**标签**：`应用` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/developer/apps/{app_id}/versions/{version}          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 获取应用版本详情信息, 如果是历史版本: 返回历史版本的全量记录；如果是未审批版本: 返回未审批版本的全量记录。特别注意： **目前只支持自建应用**。 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询应用版本信息(应用授权) `kso.app_version.read`</div></div> |


## 请求头 (Headers)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">X-Kso-Id-Type</div> | `string` | 否 | 类型<br>  * **internal** - 内部<br>  * **external** - 外部<br> | `internal`, `external` 



## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">app_id</div> | `string` | 是 | 应用id | - 
| <div style="white-space: nowrap;">version</div> | `string` | 是 | 应用版本号, 如1.0.0 | -
