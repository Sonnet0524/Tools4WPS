根据关键字召回原文相关片段(需要注意，指定scene为full_folder_recall时，可能会超时，需要使用/v7/longtask/docqa/insight/recall/rank防止超时)



**标签**：`文档片段召回能力` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/docqa/insight/recall/rank          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `POST`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 根据关键字召回原文相关片段(需要注意，指定scene为full_folder_recall时，可能会超时，需要使用/v7/longtask/docqa/insight/recall/rank防止超时) |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询和管理文档问答(应用授权) `kso.docqa.readwrite`</div><div style="margin-top: 5px;"></div><div>查询和管理智能文档库(应用授权) `kso.aidocs.readwrite`</div><div style="margin-top: 5px;"></div><div>查询和管理文档问答(用户授权) `kso.docqa.readwrite`</div><div style="margin-top: 5px;"></div><div>查询和管理智能文档库(用户授权) `kso.aidocs.readwrite`</div></div> |
