# 更新审批机器人卡片消息

    更新审批机器人已发送的审批通知消息的内容，只有审批任务类（template_id=1-5）的消息卡片需要更新（目前仅支持三方审批）

    

**标签**：`审批实例` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/workflow/message/update          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `POST`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 更新审批机器人已发送的审批通知消息的内容，只有审批任务类（template_id=1-5）的消息卡片需要更新（目前仅支持三方审批） |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询和管理审批实例(应用授权) `kso.workflow_approval_instance.readwrite`</div></div> |
