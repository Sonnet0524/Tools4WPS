# closeNavBar

## 描述
隐藏导航栏。  
> 网页应用需要完成[鉴权](/app-integration-dev/wps365/client/web-jsapi/jsapiAuth)后调用该接口。  

## 支持说明
| 客户端平台 | WPS协作版本要求 | 
|------------|----------|
| PC | &gt;&#x3D;5.24.0 |


## 输入
- 无参数
     
## 输出
- 无返回值
## 示例代码
```ts
window.ksoxz_sdk.closeNavBar()
```
