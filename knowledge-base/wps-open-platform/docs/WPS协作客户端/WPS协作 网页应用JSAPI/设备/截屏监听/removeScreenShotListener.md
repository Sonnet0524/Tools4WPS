# removeScreenShotListener

## 描述
移除截图监听。  
> 网页应用需要完成[鉴权](/app-integration-dev/wps365/client/web-jsapi/jsapiAuth)后调用该接口。  

## 支持说明
| 客户端平台 | WPS协作版本要求 | 
|------------|----------|
| iOS | &gt;&#x3D;3.14.0 |
| Android | &gt;&#x3D;3.14.0 |


## 输入
- 无参数
     
## 输出
- 无返回值
## 示例代码
```ts
window.ksoxz_sdk.removeScreenShotListener();
```
