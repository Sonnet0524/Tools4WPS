# 查询导入生成的纪要详情

查询通过导入方式创建的独立会议纪要详情，包括纪要处理状态、总结内容、章节信息等。支持通过查询参数控制是否返回完整内容。

## 请求说明

| **请求地址** | **https://openapi.wps.cn/v7/minutes/{minute_id}**                                     |
| :----------- |:--------------------------------------------------------------------------------------|
| **请求方法** | GET                                                                                   |
| **签名方式** | [KSO-1](/app-integration-dev/wps365/server/api-description/signature-description)   |
| **权限要求** | 查询会议纪要（应用授权） `kso.meeting_minutes.read`<br/>查询和管理会议纪要（应用授权） `kso.meeting_minutes.readwrite` |

## 请求头（Header）

| **Header 名称**     | **参数类型** | **是否必填** | **说明**                                                                                                     |
| :------------------ | :----------- | :----------- | :----------------------------------------------------------------------------------------------------------- |
| Content-Type        | string       | 是           | 使用：`application/json`                                                                                     |
| X-Kso-Date          | string       | 是           | RFC1123 格式的日期，例: `Wed, 23 Jan 2013 06:43:08 GMT`                                                      |
| X-Kso-Authorization | string       | 是           | KSO-1 签名值，详见[《签名方法》](/app-integration-dev/wps365/server/api-description/signature-description) |
| Authorization       | string       | 是           | 授权凭证，格式为：`Bearer {access_token}`                                                                    |

## 路径参数（Path）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**   | **参数类型** | **是否必填** | **说明**                    |
| :--------- | :----------- | :----------- | :-------------------------- |
| minute_id   | string       | 是           | 任务ID |

## 查询参数（Query）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**   | **参数类型** | **是否必填** | **说明**                    |
| :--------- | :----------- | :----------- | :-------------------------- |
| with_content   | bool       | 否           | 是否返回内容 |

## 请求地址示例

```
[GET] https://openapi.wps.cn/v7/minutes/{minute_id}?with_content=true
```

## 响应体

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**     | **参数类型** | **说明**                                                                                    |
|:-----------|:---------|:------------------------------------------------------------------------------------------|
| code       | integer  | 响应代码。非 0 表示失败，参照[《状态码说明》](/app-integration-dev/wps365/server/api-description/errorcode) |
| msg        | string   | 响应信息                                                                                      |
| data       | object   | 响应数据                                                                                      |
| ∟ id       | string   | 任务ID                                                                                       |
| ∟ status   | string   | 任务状态<br/>`running`：运行中 <br/>`failed`：失败<br/>`success`：成功                            | 
| ∟ view_url | string   | 总结查看地址                                                                                   |
| ∟ content  | object   | 内容信息（查询参数 with_content=true 才返回）                                                     |
| ∟ ∟ chapters  | object   | 章节信息                                                                                    |
| ∟ ∟ ∟ status  | string   | 章节任务状态<br/>`running`：运行中 <br/>`failed`：失败<br/>`success`：成功 
| ∟ ∟ ∟ items  | array   | 章节项列表                                                                                       |
| ∟ ∟ ∟ ∟ title  | string   | 标题                                                                                     |
| ∟ ∟ ∟ ∟ content  | string   | 内容                                                                                     |
| ∟ ∟ ∟ ∟ start_time  | int   | 开始时间（单位毫秒）                                                   |
| ∟ ∟ ∟ ∟ end_time  | int   | 结束时间（单位毫秒）                                                   |
| ∟ ∟ summary  | object   | 总结信息                                                                                       |
| ∟ ∟ ∟ content  | string   | 总结内容                                                                                     |
| ∟ ∟ ∟ status  | string   | 总结任务状态<br/>`running`：运行中 <br/>`failed`：失败<br/>`success`：成功 
 more       | object   | 更多的错误信息                                                                                    |
