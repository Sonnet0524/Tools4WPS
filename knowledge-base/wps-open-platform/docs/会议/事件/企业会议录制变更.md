# 企业会议录制变更

## 事件描述

当会议录制变更时，会向订阅方推送此事件，包含企业内所有会议录制变更事件。

## 订阅说明

| **事件主体** | **kso.meeting_recording**                                                                        |
| :----------- | :--------------------------------------------------------------------------------------------- |
| **权限要求** | 查询会议录制信息 `kso.meeting_recording.read`                                                                                             |
| **解密方式** | 详见 [解密算法](/app-integration-dev/wps365/server/event-subscription/security-verification) |

### 事件体（解密前）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**       | **参数类型** | **说明**                     |
| :------------- | :----------- |:---------------------------|
| topic          | string       | 消息主题 `kso.meeting_recording` |
| operation      | string       | 消息变更动作<br/>`ready`：完成       |
| time           | integer      | 时间（秒为单位的时间戳）               |
| nonce          | string       | iv 向量（解密时使用）               |
| signature      | string       | 消息签名                       |
| encrypted_data | string       | 消息变更的加密字段                  |

### 事件体示例（解密前）

<!-- 为了保持文档格式统一，请使用 2 个空格作为 json 的缩进符 -->

```json
{
  "topic": "kso.meeting_recording",
  "operation": "ready",
  "time": 1704074400,
  "nonce": "71***********7",
  "signature": "w6**********6Q",
  "encrypted_data": "B7**********iA=="
}
```

### 事件体数据（解密后）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**               | **参数类型** | **是否必带** | **说明**                                                   |
|:---------------------|:---------|:---------|:---------------------------------------------------------|
| meeting              | object   | 是        | 会议信息                                                     |
| ∟ id                 | string   | 是        | 会议 ID                                                    |
| ∟ subject            | string   | 是        | 会议主题                                                     |
| ∟ join_code          | string   | 是        | 入会码                                                      |
| ∟ start_time         | string   | 是        | 开始时间，unix秒级时间                                |
| ∟ end_time           | string   | 是        | 结束时间，unix秒级时间                                  |
| ∟ host_user          | object   | 是        | 主持人信息                                                    |
| ∟ ∟ type             | string   | 是        | 用户类型，`user`：wps 用户                                      |
| ∟ ∟ user_id          | string   | 是        | 用户 ID，可通过通讯录接口获取，当 type 为 user 有效                       |
| meeting_recording    | object   | 是        | 会议录制信息                                                   |
| ∟ id                 | string   | 是        | 录制 ID                                                    |
| ∟ type               | string   | 是        | 录制类型<br/>`audio`：音频文件<br/>`video`：视频文件                  |
| ∟ view_url           | string   | 否        | 录制的查看 URL                                                |
| ∟ duration           | integer  | 否        | 录制总时长（单位：秒）                                              |
