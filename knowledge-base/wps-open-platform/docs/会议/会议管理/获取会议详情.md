# 获取会议详情

获取一个会议的详细数据，若是进行中的会议则会返回入会链接



**标签**：`会议` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/meetings/{meeting_id}          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 获取一个会议的详细数据 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询会议(应用授权) `kso.meeting.read`</div><div style="margin-top: 5px;"></div><div>查询和管理会议(应用授权) `kso.meeting.readwrite`</div><div style="margin-top: 5px;"></div><div>查询会议(用户授权) `kso.meeting.read`</div><div style="margin-top: 5px;"></div><div>查询和管理会议(用户授权) `kso.meeting.readwrite`</div></div> |


## 请求头 (Headers)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">X-Kso-Id-Type</div> | `string` | 否 | 类型<br>  * **internal** - 内部<br>  * **external** - 外部<br> | `internal`, `external` 



## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">meeting_id</div> | `string` | 是 | 会议ID（视频会议的唯一标识，视频会议开始后才会产生） | -
