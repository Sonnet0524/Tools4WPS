# 资源介绍

会议是 WPS 会议系统中的核心资源，代表一次在线视频会议。会议包含会议ID、主题、参会人、时间等基本信息，参会人可通过入会码或入会链接加入。主持人拥有完整控制权，可以邀请移除参会人、设置主持人、结束会议等。一个会议可以有多个参会人、多个录制文件和纪要。

**会议属性**
- **id**: 会议ID（会议的唯一标识符）
- **subject**: 会议主题
- **join_code**: 入会码
- **join_url**: 入会链接（仅会议进行中时返回）
- **create_time**: 创建时间（Unix时间戳，秒）
- **start_time**: 开始时间（Unix时间戳，秒）
- **end_time**: 结束时间（Unix时间戳，秒）
- **host_user**: 主持人信息
- **status**: 会议状态（ongoing-进行中，ended-已结束）
