# 获取录制的章节

获取指定录制的章节



**标签**：`录制` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/meetings/{meeting_id}/recordings/{recording_id}/chapters          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 获取指定录制的章节 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询会议录制内容(应用授权) `kso.meeting_recording_content.read`</div><div style="margin-top: 5px;"></div><div>查询会议录制内容(用户授权) `kso.meeting_recording_content.read`</div></div> |




## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">meeting_id</div> | `string` | 是 | 会议ID（视频会议的唯一标识，视频会议开始后才会产生） | - 
| <div style="white-space: nowrap;">recording_id</div> | `string` | 是 | 录制ID，可通过获取录制列表接口获取 | -
