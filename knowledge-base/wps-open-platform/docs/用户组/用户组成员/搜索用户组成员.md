# 搜索用户组成员

按组成员名称筛选组中符合条件的组成员

## 请求说明

| **请求地址** | **https://openapi.wps.cn/v7/groups/{group_id}/members/search**                                                                                  |
|:---------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| **请求方法** | GET                                                                                                                                             |
| **签名方式** | [KSO-1](/app-integration-dev/wps365/server/api-description/signature-description)                                                             |
| **权限要求** | 查询和管理用户组（用户授权） `kso.group.readwrite`<br/>查询用户组（用户授权） `kso.group.read`<br/>查询和管理用户组（应用授权） `kso.group.readwrite`<br/>查询用户组（应用授权） `kso.group.read` |

## 请求头（Header）

| **Header 名称**       | **参数类型** | **是否必填** | **说明**                                                                                           |
|:--------------------|:---------|:---------|:-------------------------------------------------------------------------------------------------|
| Content-Type        | string   | 是        | 使用：`application/json`                                                                            |
| X-Kso-Date          | string   | 是        | RFC1123 格式的日期，例: `Wed, 23 Jan 2013 06:43:08 GMT`                                                 |
| X-Kso-Authorization | string   | 是        | KSO-1 签名值，详见[《签名方法》](/app-integration-dev/wps365/server/api-description/signature-description) |
| Authorization       | string   | 是        | 授权凭证，格式为：`Bearer {access_token}`                                                                 |

## 路径参数（Path）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**   | **参数类型** | **说明** |
|:---------|:---------|:-------|
| group_id | string   | 组 id   |

## 查询参数（Query）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**     | **参数类型** | **是否必填** | **说明**      |
|:-----------|:---------|:---------|:------------|
| keyword    | string   | 是        | 搜索关键字       |
| page_size  | integer  | 是        | 请求大小        |
| page_token | string   | 否        | 翻页 token    |
| with_total | boolean  | 否        | 是否返回搜索到的总条数 |

## 请求地址示例

```
[GET] https://openapi.wps.cn/v7/groups/{group_id}/members/search?keyword={string}&page_size={integer}&page_token={string}&with_total={boolean}
```
