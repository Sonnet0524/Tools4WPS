# 获取商品授权列表

查询并返回调用方身份有访问权限的商品授权列表

## 请求说明

| **请求地址** | **https://openapi.wps.cn/v7/store/entitlements**                                                                                                                                                                                                                                                                                                          |
| :----------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **请求方法** | GET                                                                                                                                                                                                                                                                                                                                                       |
| **签名方式** | [KSO-1](/app-integration-dev/wps365/server/api-description/signature-description)                                                                                                                                                                                                                                                                       |
| **权限要求** | 查询和管理租户下的所有商城权益（应用授权） `kso.store_allocation.readwrite.all`<br/>查询和管理该应用权限下的商城权益（应用授权） `kso.store_allocation.readwrite.ownedby`<br/>查询租户下的所有商城权益（应用授权） `kso.store_allocation.read.all`<br/>查询该应用权限下的商城权益（应用授权） `kso.store_allocation.read.ownedby` |

## 请求头（Header）

| **Header 名称**     | **参数类型** | **是否必填** | **说明**                                                                                                     |
| :------------------ | :----------- | :----------- | :----------------------------------------------------------------------------------------------------------- |
| Content-Type        | string       | 是           | 使用：`application/json`                                                                                     |
| X-Kso-Date          | string       | 是           | RFC1123 格式的日期，例: `Wed, 23 Jan 2013 06:43:08 GMT`                                                      |
| X-Kso-Authorization | string       | 是           | KSO-1 签名值，详见[《签名方法》](/app-integration-dev/wps365/server/api-description/signature-description) |
| Authorization       | string       | 是           | 授权凭证，格式为：`Bearer {access_token}`                                                                    |

## 请求地址示例

```
[GET] https://openapi.wps.cn/v7/store/entitlements
```

## 响应体

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**         | **参数类型**  | **说明**                                                                                                                           |
| :--------------- | :------------ |:---------------------------------------------------------------------------------------------------------------------------------|
| code             | integer       | 响应代码。非 0 表示失败，参照[《状态码说明》](/app-integration-dev/wps365/server/api-description/errorcode)                                        |
| msg              | string        | 响应信息                                                                                                                             |
| data             | object        | 响应数据                                                                                                                             |
| ∟ items          | array[object] | 商品授权列表                                                                                                                                |
| ∟ ∟ allocated    | integer       | 已分配数量                                                                                                                            |
| ∟ ∟ can_allocate | boolean       | 是否能分配                                                                                                                            |
| ∟ ∟ desc         | string        | 授权中文名称                                                                                                                           |
| ∟ ∟ expire_at    | integer       | 授权到期时间                                                                                                                           |
| ∟ ∟ key          | string        | 授权 key                                                                                                                           |
| ∟ ∟ name         | string        | 授权英文名称，当前被授权的商品规格名称<br/>如果是商品类型是 `service_package`，该字段表示套餐名；<br/>如果商品类型是 `priv_package`，该字段表示权益包名；<br/>如果商品类型是 `sku`，该字段表示 skuid |
| ∟ ∟ product_type | string[enum]  | 授权的商品类型 <br/>`service_package`：套餐；`privilege_package`：权益包；`sku`：标准商品                                                             |
| ∟ ∟ start_at     | integer       | 授权开始生效时间                                                                                                                         |
| ∟ ∟ total        | integer       | 授权总量                                                                                                                             |
| ∟ total          | integer       | 数据总量                                                                                                                             |
| more             | object        | 更多的错误信息                                                                                                                          |
