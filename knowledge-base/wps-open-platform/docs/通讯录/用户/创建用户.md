# 创建用户

创建新用户

## 请求说明

| **请求地址** | **https://openapi.wps.cn/v7/users/create**                                          |
| :----------- | :---------------------------------------------------------------------------------- |
| **请求方法** | POST                                                                                |
| **签名方式** | [KSO-1](/app-integration-dev/wps365/server/api-description/signature-description) |
| **权限要求** | 查询和管理通讯录信息（应用授权） `kso.contact.readwrite`                            |

## 请求头（Header）

| **Header 名称**     | **参数类型** | **是否必填** | **说明**                                                                                                     |
| :------------------ | :----------- | :----------- | :----------------------------------------------------------------------------------------------------------- |
| Content-Type        | string       | 是           | 使用：`application/json`                                                                                     |
| X-Kso-Date          | string       | 是           | RFC1123 格式的日期，例: `Wed, 23 Jan 2013 06:43:08 GMT`                                                      |
| X-Kso-Authorization | string       | 是           | KSO-1 签名值，详见[《签名方法》](/app-integration-dev/wps365/server/api-description/signature-description) |
| Authorization       | string       | 是           | 授权凭证，格式为：`Bearer {access_token}`                                                                    |

## 请求体（Body）

<!-- 为了保持文档格式统一：（1）参数说明中请使用中文全角标点符号；（2）中英文间请添加空格 -->

| **名称**            | **参数类型**      | **是否必填** | **说明**                                     |
|:------------------|:--------------|:---------|:-------------------------------------------|
| city              | string        | 否        | 城市                                         |
| country           | string        | 否        | 国家                                         |
| dept_ids          | array[string] | 是        | 加入的部门 id                                   |
| email             | string        | 否        | 邮箱                                         |
| employee_id       | string        | 否        | 工号                                         |
| employer          | string        | 否        | 就职单位                                       |
| employment_status | string        | 否        | 员工状态。<br/> `probationary`:试用。                                       |
| employment_type   | string        | 否        | 员工类型。<br/>普通企业：`permanent`: 正式； `intern`:  实习。<br/> 教育企业：`teacher`:教师；`student`:学生；`staff`:教职工；`employee`:聘用人员；`alumni`:校友；`other`:其他                                        |
| gender            | string[enum]  | 否        | 性别 <br/>`male`：男性；`female`：女性；`secrecy`：保密 |
| leader_id         | string        | 否        | 直属主管的用户ID                                   |
| phone             | string        | 否        | 手机号码                                       |
| telephone         | string        | 否        | 座机                                         |
| title             | string        | 否        | 职务信息                                       |
| user_name         | string        | 是        | 用户名称                                       |
| work_place        | string        | 否        | 办公地点                                       |
| def_dept_id       | string        | 否        | 主部门id                                      |
| ex_user_id        | string        | 否        | 外部身份源ID                                    |

## 请求地址示例

```
[POST] https://openapi.wps.cn/v7/users/create
```

| ∟ title             | string        | 职务信息                                                                                                         |
| ∟ user_name         | string        | 用户名称                                                                                                         |
| ∟ work_place        | string        | 办公地点                                                                                                         |
