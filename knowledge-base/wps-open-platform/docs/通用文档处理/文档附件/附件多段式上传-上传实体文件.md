# 上传实体文件

上传文件分为三个步骤：

1、使用[申请上传地址](https://open.wps.cn/api-studio/doc-station/1?doc=multiupload-1-get-upload-address)接口，获取上传文件所需信息；

**2、通过接口响应体的上传信息，构建 HTTP 请求，上传实体文件到云存储；**

3、使用[提交上传完成](https://open.wps.cn/api-studio/doc-station/1?doc=multiupload-3-complete-upload)接口，提交上传完成信息。

| **HTTP 方法** | [申请上传地址](https://open.wps.cn/api-studio/doc-station/1?doc=multiupload-1-get-upload-address)接口返回的方法 method                                                                                                                                                                                                                                                                                                                                                                                                     |
| **接口描述**    | 无                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **签名方式**    | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description)                                                                                                                                                                                                                                                                                                                                                                                               |
| **限频策略**    | 无                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **权限要求**    | 文档管理能力(应用授权) <span style="background-color: rgba(142, 150, 170, 0.14);"><span style="color: rgb(52, 81, 178);"><span style="font-family: monospace, monospace;"><span style="font-size: 12.25px;">`kso.documents.readwrite`</span></span></span></span>文档管理能力(用户授权) <span style="background-color: rgba(142, 150, 170, 0.14);"><span style="color: rgb(52, 81, 178);"><span style="font-family: monospace, monospace;"><span style="font-size: 12.25px;">`kso.documents.readwrite`</span></span></span></span> |



## 请求头（Header）

更多详细响应体参数信息见：[申请上传地址的响应体](https://open.wps.cn/api-studio/doc-station/1?doc=multiupload-1-get-upload-address#%E5%93%8D%E5%BA%94%E4%BD%93response)



## 请求体（Body）

上传的文件内容（文件流）。



## 响应体（Response）

各存储返回结果不一致，请根据实际存储寻找下一步需要的参数，`etag` , `key` 值一般在 header 中。
