# 获取知识库成员列表



**标签**：`知识库` 


## 请求说明
| 字段          | 值                  |
|--------------|---------------------|
| **<div style="white-space: nowrap;">请求地址</div>**  | https://openapi.wps.cn/v7/wiki/spaces/{space_id}/members          |
| **<div style="white-space: nowrap;">HTTP 方法</div>** | `GET`    |
| **<div style="white-space: nowrap;">接口描述</div>**  | 获取知识库成员列表 |
| **<div style="white-space: nowrap;">签名方式</div>**  | [KSO-1](https://open.wps.cn/documents/app-integration-dev/wps365/server/api-description/signature-description) |
| **<div style="white-space: nowrap;">限频策略</div>** | 无 |
| **<div style="white-space: nowrap;">权限要求</div>** | <div><div>查询和管理知识库(用户授权) `kso.wiki.readwrite`</div></div> |




## 路径参数 (Path)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">space_id</div> | `string` | 是 | 知识库id | - 



## 查询参数 (Query)
| 属性名 | 类型 | 是否必填 | 描述 | 可选值 |
|-------|------|------|-------|-------|
| <div style="white-space: nowrap;">page_size</div> | `integer` | 否 | 分页大小，范围1~100，默认30 | - 
| <div style="white-space: nowrap;">page_token</div> | `string` | 否 | 分页标记 | - 
| <div style="white-space: nowrap;">roles</div> | `array` | 否 | 成员角色 | `readonly`, `normal`, `admin`, `owner`
